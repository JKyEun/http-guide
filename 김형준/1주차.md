# 01. HTTP 개관 ~ 02. URL과 리소스

## 문제

HTTP 통신에서 터널의 역할로 가장 적절한 설명을 고르세요.

1. 클라이언트의 요청을 대신 서버로 전달하며, 요청과 응답을 필터링하는 서버
2. 자주 요청되는 문서를 저장해두었다가 다시 요청 시 빠르게 제공하는 서버
3. 서로 다른 프로토콜 간의 변환을 수행하여 통신을 중개하는 서버
4. 두 연결 사이에서 데이터를 열어보지 않고 그대로 전달하는 서버

## 정답

4번

## 해설

터널은 두 커넥션 사이에서 데이터를 그대로 전달하는 HTTP 애플리케이션이다.  
주로 SSL/TLS 트래픽을 HTTP 연결을 통해 그대로 전달할 때 사용됨.  
대표적인 예로 HTTP/SSL 터널링이 있으며, 이 방식은 방화벽을 통과해 암호화된 트래픽을 안전하게 전송할 수 있게 한다.
